Tags e Formatações de Texto

Semânticas e Não Semânticas

Semânticas

  • strong para deixar em negrito
  • em para deixar em itálico
  • mark para marcar
  • small para deixar a letra pequena
  • del para linha no meio do texto
  • ins para sublinhar
  • sup para sobrescrito
  • sub para subscrito

Não Semânticas

  • b para negrito
  • i para itálico
  • big para texto um pouco maior
  • u para sublinhar

Código-fonte / Pré-formatação (usando a tag code)

O comando document.getElementById('teste') é escrito em linguagem de JavaScript. (sem a tag code)

O comando document.getElementById('teste') é escrito em linguagem de JavaScript. (com a tag code)

Exemplo de Código em Python

sem a tag pre

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR") print('Fim do programa')

com a tag pre

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR")
    print('Fim do programa')
